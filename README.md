# Computational Practicum

A GUI App for plotting numerical methods of solving Differential Equations.

- [x] Euler Method
- [x] Improved Euler Method
- [x] Runge-Kutta Method
## Stack:

- [x] OpenGL 3.3

  - [x] GLAD
- [x] DearImGui
- [x] ImPlot
